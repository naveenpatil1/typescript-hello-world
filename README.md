# TypeScript Hello World

A simple Hello World application written in TypeScript.

## Setup

Install dependencies:

```bash
npm install
```

## Running the Application

### Option 1: Run directly with ts-node (Development)

```bash
npm run dev
```

### Option 2: Compile and run (Production)

```bash
npm run build
npm start
```

## Project Structure

- `index.ts` - Main TypeScript source file
- `tsconfig.json` - TypeScript configuration
- `package.json` - Project dependencies and scripts
- `dist/` - Compiled JavaScript output (created after build)
